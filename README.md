# AdvogadoTemplates

Templates de site para advocacia brasileira, construídos a partir da análise de escritórios reais (Mattos Filho, TozziniFreire, Demarest, Lefosse, Galvão & Silva).

**Demo ao vivo:** [vejaseusite.github.io/AdvogadoTemplates](https://vejaseusite.github.io/AdvogadoTemplates/)

## Templates disponíveis

| # | Nome | Nicho | Stack |
|---|---|---|---|
| 01 | `premium-corporate` | Full-service PJ, M&A, tributário empresarial | HTML + CSS, Fraunces + Inter |
| 02 | `seo-first` | Pessoa física, alto volume, conversão orgânica | HTML + CSS, Inter |
| 03 | `solo-boutique` | Advogado individual, nicho editorial | HTML + CSS, Cormorant Garamond + Inter |

## Como usar

```bash
gh repo clone VejaSeuSIte/AdvogadoTemplates meu-site
cd meu-site/templates/<escolha-um>
# edite o index.html com seu nome, OAB, endereço
```

Deploy direto no GitHub Pages — zero build, zero dependências.
